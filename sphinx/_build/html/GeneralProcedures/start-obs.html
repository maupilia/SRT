Start the observations

$ : commands to insert in a shell

> : commands to insert in the operatorInput panel

: check the execution on the monitor

On DISCOS-CONSOLE

  1. Insert your project number

    > project=[projectID] Fig.21

  2. Initial setup

    > antennaReset Fig.37

    > setupCCB Fig.18 Fig.30

    > goTo=*,85d Fig.15

  3. Follow the link below to perform the pointing optimization:

    Pointing and Focus optimization

On PULSAR/VLBI

  1. Login as pulsar. Place your schedules in folder /home/pulsar/scheds/[your project code]

  2. From a terminal, start SEADAS

    $ seadas

  3. On another workspace, open a new terminal and start a vnc session for corr@psrdfb:

    $ vncviewer psrdfb:2

  4. Type password at prompt.

In the vnc session corr@psrdfb

  1. Open a terminal and check the system clock is working properly (time properly synchronized, tick phase ~ 100 ns, positive or negative) by typing

    $ atdc

  2. If DFBCONTROLLER is not already on, open a new terminal and start it

    $ dfbcontroller

  3. In DFBCONTROLLER window, check that the colored label at the right of the “tkds” label is green and displays the word “CONNECTED”. If not, click on it.

Inside SEADAS window

  1. Enable the antenna control by clicking on the red label at the top right corner of the Antenna and Pointing Management frame. Fig.31

  2. Check that the label becomes green and displays the word “ENABLED” Fig.32

  3. Select “Schedule” in the “Session mode” combo box Fig.33

  4. Click “Schedule Management”. A window named “Seadas schedule management” pops up Fig.34

  5. Click “Load sched” in “Schedule management” window. A system window pops up for browsing system directories and selecting the schedule file

  6. In “Schedule manager” window select schedule lines to be done - mouse left click on each single line - or click button “Select all” for loading the entire schedule in the “Observations List” window Fig.35

  7. If necessary, rearrange the order of the observations in the “Observations List” window - left button click == cut line ; mid button click == paste line

  8. Back on SEADAS main window click “Observe”

  9. Do not forget to verify and adjust attenuation levels. You can find the test schedule TEST.scd under the directory scheds/ (or you can add a similar line to your schedule). Once you have started the schedule, check the PDFB3 panel in the psrdfb vnc session, Under tab Samplers where you fill find the the RMS levels of the channels. Also, check the SPD plotting tool to compare the bandpasses of the two channel. Aim to align the bandpasses, with the highest channel RMS around 13. In order to do so, iteratively go back to the SEADAS interface and select the desired attenuation values (which can be different for the two channels), then press set atten.
