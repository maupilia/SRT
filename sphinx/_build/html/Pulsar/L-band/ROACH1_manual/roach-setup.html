Roach setup

Initial setup for the LEAP cluster.

In the VNC

  1. Open a window (W1) with 8 tabs. Each tab will be connected to each of the 8 cluster nodes and will be used to launch the daemons. In the first tab, type:

    $ ssh -X node1

    $ ssh -X node2 in the second tab, etc… for the 8 tabs.

  2. Open a second window (W2) and go to :

    $ ~/externalClient/Control/

    This window will be used to start and stop data acquisition.

  3. In W2: type a command here to select L-band (and not P-band):

    $ ~/roach/scripts/setLband.sh

  4. In W2: type a command here to select baseband mode:

    $ ~/roach/scripts/baseband.sh

  5. In W2: type a command here to select folding mode:

    $ ~/roach/scripts/folding.sh

  6. Open a third window (W3) with 8 tabs (this will be used to look at the results). As before, type $ ssh -X node1, $ ssh -X node2, etc. for each tab.
